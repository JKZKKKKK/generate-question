# <a id="zh"></a>繁體中文 (Traditional Chinese)

這是一個專為「日本地標衛星空照圖」設計的高階自動化測資生成與防禦型 AI 裁判訓練流水線。本專案不僅包含資料擴增，更整合了高階特徵分析、以及能自動剔除「一眼假」廢圖的強健訓練機制。

## 📂 核心模組詳細介紹 (Modules in Detail)

本專案將複雜的流程拆分為多個獨立模組，確保程式碼的可維護性：

### 1. 🚀 自動化流水線 (Automation Pipelines)
* **`auto_gen_test_data.py` (一鍵測資生成打包)**
  * **功能**：負責自動化「出考卷」的所有流程。
  * **亮點**：內建防呆導航與大掃除機制（自動處理 Windows 唯讀檔案刪除報錯）。它會依序呼叫核心生成器與展平工具，並自動將最終測資打包成 `fin_test.zip` 與 `test.zip`。
* **`auto_pipline.py` (一鍵裁判訓練)**
  * **功能**：負責 AI 裁判的自動化訓練準備。
  * **亮點**：執行嚴格的環境清理（徹底清除舊的假圖與隔離區），接著呼叫假圖採樣腳本，最後啟動裁判訓練，並在結尾輸出總耗時。

### 2. 🧠 核心處理引擎 (Core Engines)
* **`generate_test_data.py` (測資生成引擎)**
  * **功能**：讀取 `config.json` 設定，對原始圖片進行各種干擾與擴增。
  * **亮點**：支援多進程 (Multiprocessing) 加速。內建完美無黑邊旋轉、空間扭曲 (Shear)、色彩偏移、雜訊、模糊、馬賽克與遮擋等「十全大補」技能，高度自訂化。
* **`train_discriminator.py` (防禦型 AI 裁判訓練)**
  * **功能**：訓練一個圖片二元分類器 (真/假)。
  * **亮點**：引入了 **「廢圖自動隔離機制 (Bad-Data Quarantine)」**。在訓練過程中，若模型以 >90% 的信心判定某張圖為假圖，代表該圖破綻太大，系統會動態將其移至 `bad_data_quarantine` 資料夾，避免模型學到過於簡單的特徵。

### 3. 🔍 分析與資料處理工具 (Utilities)
* **`color_analyzer.py` (終極特徵分析報告)**
  * **功能**：量化比較官方真圖與您生成的假圖。
  * **亮點**：使用 OpenCV (Canny) 進行邊緣密度分析，並計算極暗 (dark_ratio) 與極亮 (highlight_ratio) 區域。程式最終會給出具體的戰術建議（例如：陰影缺乏，建議開啟 config 中的 shadows）。
* **`flatten_and_rename.py` (資料夾展平與重命名)**
  * **功能**：將帶有標籤的樹狀資料夾轉換為單一資料夾結構。
  * **亮點**：支援設定隨機種子 (Seed) 確保實驗可重現性。自動從 1 開始重新命名，並同步產出對應的 `mapping.csv`，方便後續對答案。

# <a id="jp"></a>日本語 (Japanese)
# 🤖 AI 甲子園 - テストデータ生成および識別器トレーニングパイプライン

本プロジェクトは、「日本のランドマーク衛星写真」向けに設計された、高度なテストデータ自動生成および防御型AI識別器（Discriminator）のトレーニングパイプラインです。データ拡張だけでなく、高度な特徴分析や、「一目で偽物とわかる」粗悪な画像を自動排除する堅牢な学習メカニズムを統合しています。

## 📂 モジュールの詳細 (Modules in Detail)

保守性を確保するため、複雑な処理を複数の独立したモジュールに分割しています。

### 1. 🚀 自動化パイプライン (Automation Pipelines)
* **`auto_gen_test_data.py` (テストデータの一括生成とパッケージング)**
  * **機能**：テスト問題作成の全プロセスを自動化します。
  * **特徴**：ディレクトリの自動移動や、以前のデータのクリーンアップ機能（アクセス拒否エラーの回避処理を含む）を内蔵。生成からフラット化、ZIP圧縮（`fin_test.zip`, `test.zip`）まで一貫して行います。
* **`auto_pipline.py` (識別器の全自動トレーニング)**
  * **機能**：AI識別器の学習準備から実行までを自動化します。
  * **特徴**：古い偽画像や隔離フォルダを完全に初期化し、サンプリングスクリプトを実行後、トレーニングを開始します。最後に全体の処理時間を出力します。

### 2. 🧠 コアエンジン (Core Engines)
* **`generate_test_data.py` (テストデータ生成エンジン)**
  * **機能**：`config.json` を読み込み、元画像に対して様々なノイズや拡張を適用します。
  * **特徴**：マルチプロセス (Multiprocessing) に対応。黒枠を出さない完全な回転、空間の歪み(Shear)、色調整、ノイズ、ぼかし、モザイク処理など、多岐にわたる加工スキルを高度にカスタマイズ可能です。
* **`train_discriminator.py` (防御型AI識別器トレーニング)**
  * **機能**：画像が本物か偽物かを判定する二値分類器を学習させます。
  * **特徴**：**「粗悪データ自動隔離メカニズム (Bad-Data Quarantine)」** を導入。学習中に、モデルが90%以上の確度で「偽物」と判定した画像は特徴が単純すぎるとみなし、動的に `bad_data_quarantine` フォルダへ移動させ、モデルの汎化性能を向上させます。

### 3. 🔍 分析・ユーティリティツール (Utilities)
* **`color_analyzer.py` (特徴分析・レポートツール)**
  * **機能**：公式の本物画像と、生成した偽物画像を定量的に比較します。
  * **特徴**：OpenCV (Canny) を用いたエッジ密度分析や、極端な暗部/明部の割合を計算。その結果に基づき、「影が足りないため config で shadows を有効にしてください」などの具体的な改善アドバイスを自動出力します。
* **`flatten_and_rename.py` (階層フラット化・リネームツール)**
  * **機能**：サブフォルダに分かれた画像を一つのフォルダにまとめます。
  * **特徴**：ランダムシードによる再現性の確保。画像を1から連番でリネームし、元のファイルパスとの対応表 (`mapping.csv`) を同時に生成します。
